Requirements - 
1. Eclipse
2. JRE 1.8
3. Apache tomcat 8

Steps to use this code - 

1. Go to Files on Eclipse and click on Import.
2. Select "Existing project into workspace".
3. Choose the project and click on Finish.
4. Go to "Configure build path" then select "Project facets" then on the right hand side select "Runtimes". Check the Apache tomcat option and click on Apply button.
5. Deploy the bot locally and add the callback URL. Read this guide - https://www.gupshup.io/developer/docs/bot-platform/guide/deploying-your-bot-locally

The callback URL should be like - https://abc/currencyConverterBot/currencyConverter
